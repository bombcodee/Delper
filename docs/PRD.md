# Delper - Product Requirements Document

## 제품 개요
- **이름**: Delper (Develop + Helper)
- **타입**: 정적 웹 가이드 (HTML/CSS/JS, 서버 없음)
- **대상**: 1인 개발자 (기획~운영까지 혼자 하는 사람)
- **목적**: 개발 프로세스 전반의 용어, 전략, 기술, 운영 지식을 한 곳에 정리한 올인원 가이드
- **슬로건**: The All-in-One Dev Guide
- **배포**: GitHub Pages (`https://<username>.github.io/Delper/`)

## 핵심 가치
1. **한 곳에서 전부**: 기획→개발→출시→운영 전 과정 커버
2. **실용적**: 이론보다 "1인 개발자가 실제로 뭘 해야 하는지"에 집중
3. **빠른 접근**: Ctrl+K 커맨드 팔레트로 모든 콘텐츠 즉시 검색
4. **조립식 구조**: 데이터/디자인/기능이 분리되어 쉽게 수정 가능

## 주요 기능
- **테마 5종**: Dark, Midnight, Spring, Blossom, Sky
- **Ctrl+K 커맨드 팔레트**: 용어/기술/섹션/산업 통합 검색
- **F1 도움말**: 단축키 안내
- **용어집 필터**: 카테고리별 + 즐겨찾기 필터 (개수 표시)
- **북마크**: 용어 즐겨찾기 (리본 UI, localStorage 저장)
- **용어 상호링크**: 설명에서 다른 용어 클릭 시 이동
- **체크리스트**: 클릭 시 취소선으로 완료 표시
- **산업분야 치트키**: 단계별 핵심 팁 토글
- **인쇄 스타일**: @media print 지원
- **반응형**: 모바일 사이드바 (햄버거 메뉴)

## 기술 스택
- HTML5 (시맨틱 마크업)
- CSS3 (커스텀 프로퍼티, 그리드, 플렉스)
- Vanilla JS (프레임워크 없음, 글로벌 함수)
- Google Fonts (Noto Sans KR, JetBrains Mono)
- GitHub Pages (배포)

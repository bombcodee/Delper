### 주간 자동 최신화 시스템
> 매주 1회 자동으로 Delper 콘텐츠를 최신 상태로 유지하는 파이프라인

**대상 영역:**
- [ ] 용어집 — 새로운 개발 용어/트렌드 용어 추가 (예: AI Agent, MCP, vibe coding 등)
- [ ] 산업분야 가이드 — 각 분야별 최신 기술 스택, 법적 요건, 성장 전략 업데이트
- [ ] 기술 스택 — 새로운 프레임워크/도구 추가, 단종된 도구 표시
- [ ] 추천 리소스 — 성공 사례 추가, 깨진 링크 자동 감지 및 교체, 새로운 유용 도구 반영
- [ ] 성공 사례/최신 트렌드 — 1인 개발자 성공 사례, 최신 동향, 수익화 트렌드 반영

**구현 방식 (검토 중):**
- GitHub Actions 스케줄 (cron: 매주 월요일)
- Claude API 또는 웹 크롤링으로 최신 정보 수집
- 수집된 내용을 data/ 폴더의 JS 파일에 자동 반영
- PR 자동 생성 → 사람이 리뷰 후 머지 (완전 자동 배포는 위험)
- 깨진 링크 체커 (모든 외부 URL을 주기적으로 검증)

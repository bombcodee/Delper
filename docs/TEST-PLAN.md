# Delper - 테스트 계획

## 수동 테스트 체크리스트

### 테마 전환
- [ ] Dark → Midnight → Spring → Blossom → Sky 순서로 전환
- [ ] 각 테마에서 텍스트 가독성 확인
- [ ] 각 테마에서 카드, 뱃지, 태그 색상 조화 확인
- [ ] 새로고침 후 선택한 테마 유지되는지 확인
- [ ] 사이드바 로고 그라데이션이 테마별로 변하는지 확인

### 용어집
- [ ] 용어 150개 전부 렌더링 되는지 확인
- [ ] 카테고리 필터 클릭 → 해당 카테고리만 표시
- [ ] 필터 버튼 개수 표시 정확한지 확인
- [ ] 검색창에 "PRD" 입력 → 관련 용어만 필터링
- [ ] 검색 + 필터 조합 동작 확인
- [ ] 용어 설명 내 다른 용어 링크 클릭 → 해당 용어로 스크롤
- [ ] 즐겨찾기 리본 클릭 → 앰버색 변경
- [ ] "★ 즐겨찾기" 필터 → 북마크된 용어만 표시
- [ ] 즐겨찾기 개수 실시간 업데이트
- [ ] 새로고침 후 즐겨찾기 유지

### 커맨드 팔레트
- [ ] Ctrl+K → 팔레트 열림
- [ ] 팔레트 열면 북마크된 항목이 상단에 표시
- [ ] 검색어 입력 → 용어/기술/섹션/산업 통합 검색
- [ ] ↑↓ 키보드 네비게이션
- [ ] Enter → 선택 항목으로 이동
- [ ] Esc → 팔레트 닫힘
- [ ] 오버레이 클릭 → 팔레트 닫힘

### 산업분야
- [ ] 5개 산업 탭 전환 (웹앱/게임/자동화/모바일/콘텐츠)
- [ ] 예정 산업 탭은 비활성 상태
- [ ] 단계 클릭 → 아코디언 열림
- [ ] 치트키 토글 → 열린 아코디언 안에 치트카드 표시
- [ ] 치트키 ON 상태에서 단계 열기/닫기 정상 동작

### AI & 자동화 섹션
- [ ] 3개 탭 전환 (바이브코딩 / AI 활용 / 하네스 엔지니어링)
- [ ] 기본 탭(바이브코딩) 활성화 상태로 로드
- [ ] 탭 클릭 → 해당 패널만 표시, 나머지 숨김
- [ ] 각 패널 내 아코디언 열기/닫기 정상 동작
- [ ] 탭 활성 상태 스타일 (.active) 시각적으로 구분
- [ ] 각 테마(5종)에서 탭/패널 색상 가독성 확인

### 체크리스트
- [ ] 체크박스 클릭 → 취소선 + 흐림
- [ ] 문장 아무 곳이나 클릭 → 동일 동작
- [ ] 재클릭 → 취소선 해제

### 전략 가이드
- [ ] 퍼널 (사용자 여정, AARRR) 단계 구분 명확한지
- [ ] 퍼널 텍스트 한 줄에 들어가는지 (줄바꿈 없음)
- [ ] KPI 테이블 건강한 수치(초록), 위험 신호(빨강)
- [ ] 가격표 "인기" 뱃지 잘림 없는지
- [ ] 의사결정 트리 4칸 정렬

### 네비게이션
- [ ] 사이드바 10개 섹션 전부 클릭 → 해당 섹션 표시
- [ ] F1 → 도움말 오버레이
- [ ] 모바일(768px 이하) → 햄버거 메뉴 동작

### 인쇄
- [ ] Ctrl+P → 사이드바/네비 숨김, 콘텐츠만 인쇄
- [ ] 아코디언 인쇄 시 전부 열린 상태

### 반응형
- [ ] 데스크탑 (1200px+): 사이드바 고정, 콘텐츠 중앙
- [ ] 태블릿 (768px): 사이드바 접힘, 그리드 2열
- [ ] 모바일 (480px): 그리드 1열, 풀 와이드

### 추천 리소스
- [ ] 모든 외부 링크 클릭 → 새 탭에서 열리는지
- [ ] 링크 URL이 유효한지 (404 없는지)
